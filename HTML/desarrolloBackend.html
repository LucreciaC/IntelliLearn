Web Development 101

Es un curso introductorio diseñado para quienes desean dar sus primeros pasos en el mundo del desarrollo web. A lo largo del curso, aprenderás los fundamentos de la programación web y las tecnologías clave que conforman los sitios web modernos.

Instructor certificado: por la Universidad Champagnat
Que aprenderas? HTML: La estructura básica de una página web. Aprenderás a crear y organizar contenido usando etiquetas HTML. CSS: El lenguaje para dar estilo a tus sitios web. Descubrirás cómo aplicar colores, fuentes, diseños y cómo hacer tus páginas atractivas y profesionales. JavaScript: Introducción a la programación del lado del cliente. Aprenderás a hacer tus páginas interactivas y dinámicas, mejorando la experiencia del usuario. Git & GitHub: Los conceptos básicos de control de versiones para que puedas colaborar en proyectos y mantener un historial de tus cambios. Responsive Design: Cómo hacer que tus páginas web se vean bien en dispositivos de todos los tamaños, desde smartphones hasta computadoras de escritorio
¿A quién está dirigido? Este curso está diseñado para principiantes absolutos, sin necesidad de experiencia previa en programación. Es perfecto para quienes quieren aprender a construir sitios web desde cero y adquirir las habilidades básicas para avanzar en el mundo del desarrollo web.
Requisitos previos Solo necesitas una computadora, acceso a Internet y ganas de aprender. ¡El resto lo aprenderás durante el curso!
